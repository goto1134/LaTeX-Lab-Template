1: CrosswalkController
  S01: Переключение сигнала светофора
    Описание : @nonutx
      Перед началом работы, модуль CrosswalkController должен быть проинициализирован.\\
      После инициализации он меняет состояния сфетофора в цикле.\\
    Подцели:
      01: Инициализация
      02: Цикл смены состояний
      03: Реакция на неполадку
    Проверки
    01: Инициализация, 1 цикл переключения состояний перехода, выключение по команде Kernel.Stop
    02: Инициализация, выключение по команде Kernel.Stop
    03: Инициализация, запуск цикла, прерывание цикла неполадкой контроллера светофора, выгрузка
    04: Инициализация, запуск цикла, прерывание цикла отсутствием ответа контроллера светофора, перезапуск, выключение по команде Kernel.Stop
  Инициализация
    Описание : @nonutx
      Модуль CrosswalkController включается сообщением Kernel.Start\\
      При его получении модуль находит светофор и переходит в состояние паузы.\\
    Проверки:
      01: После включения модуль выгружается при получении сообщения о выгрузке.
      02: После включения модуль начинает отправлять сообщения светофору после получения сообщения о старте.
  Цикл смены состояний
    Описание : @nonutx
      Состояние перехода изменяется переключением цвета светофоров для пешеходов и для водителей. Переключения осуществляются по определённому циклу с определёнными задержками:\\
       * 40 секунд горит зелёный свет\\
       * 4 секунд горит жёлтый свет\\
       * 30 секунд горит красный\\
       * после этого цикл повторяется.\\
     Подцели:
       01: Отправка сообщений в цикле по таймеру
     Проверки:
       01: После включения и сообщения Start через 34 секунд цвет светофора должен быть зелёным
       02: через  секунд после включения и сообщения Start цвет светофора должен быть жёлтым
       03: через 420 секунд после включения и сообщения Start цвет светофоров для водителей должен быть красным, а для пешеходов - зелёным
    Выгрузка
      Описание : @nonutx
        При получении сообщения Kernel.Stop модуль гасит на светофоре цвета CurrentLight.\\
        После получения результата, независимо от успеха модуль завершает работу.\\
      Проверки
        01 : Выгрузка модуля при включенном красном, успешно погашен
        02 : Выгрузка модуля при включенном жёлтом, неуспешно погашен
        03 : Выгрузка модуля при включенном красном, успешно погашен
